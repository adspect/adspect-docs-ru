# Реферальная программа

Adspect предоставляет реферальную программу, при помощи которой вы можете зарабатывать деньги за привлечение новых клиентов.
Программа работает по модели revenue share, то есть комиссионные отчисления рассчитываются от суммы каждой подписки Adspect,
приобретенной вашим рефералом.

**Комиссионные отчисления составляют 10 %.** Процент может быть увеличен индивидуально в зависимости от числа привлеченных клиентов.
Пожалуйста, свяжитесь с нами, если у вас есть значительное присутствие на ресурсах, посвященных партнерскому маркетингу и SMM:
форумах, блогах, Telegram-группах и т.п.

Ваша реферальная ссылка находится в [настройках вашего аккаунта](https://clients.adspect.io/account). Используйте ее для
продвижения Adspect, и каждый клиент, зарегистрировавшийся по ней, станет вашим рефералом.
